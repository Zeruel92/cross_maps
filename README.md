# cross_maps

A new Flutter project.

## Installing

- Add into pubspec.yaml 
```yaml
cross_picker:
    git:
      url: https://github.com/pspgt/cross_maps.git
```
- run `flutter pub get`
- Now import with `import 'package:cross_maps/cross_maps.dart';`

# Example 

```dart
CrossMaps crossMaps = CrossMaps();
crossMaps.getMaps(lat,lng);
```
